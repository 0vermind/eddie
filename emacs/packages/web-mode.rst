keybindings
===========

```
C-c C-n  # web-mode-navigate
C-c C-f  # web-mode-fold-or-unfold
C-c C-s  # web-mode-snippet-insert
C-c C-m  # web-mode-mark-expand
C-c C-w  # web-mode-whitespaces-show

M-;      # web-mode-comment-or-uncomment
```
