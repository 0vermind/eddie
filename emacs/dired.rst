============
Dired Plugin
============

------------
Key Bindings
------------

C-x d open dir dired mode
M-x find-dired
M-x find-name-dired

s - sort files
C-u s - sort by format options

RET - enter into dir
^ - go up one level
i - insert dir into current buffer
</> - navigate dirs
M-C-n/p - navigate buffers
$ - fold the buffer
M-$ - fold all buffers

T - touches file
m - mark file
u - unmark file
U - unmark all files
R - rename file
g - refresh buffer
S - sym link
d - mark files for deletion
x - delte martked files
z - compress files
Z - compress all files

M-s a C-s - incremental search on selected files
