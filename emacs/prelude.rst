Key bindings
------------

C-x m  eshell  # launch inferior shell
C-x M-m  shell  # launch default shell
C-h A  apropos  # patterns matching word
C-c j  ace-jump-word-name
C-c J  ace-jump-buffer
C-c a  prelude-move-beginning-of-line
